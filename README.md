#  Project Management & Task Tracking System

A full-stack Project Management and Task Tracking System designed to help me efficiently manage projects, assign tasks, track progress, and meet deadlines.
This project was developed with a focus on clean architecture, usability, and real-world applicability.

---

## Project Overview

Managing multiple projects and tasks manually can be inefficient and error-prone.  
This system provides a **centralized platform** where users can:

- Create and manage projects
- Assign tasks to users
- Track task status and progress
- Monitor deadlines
- Improve productivity and collaboration

The application follows a **clientâ€“server architecture**, separating frontend and backend for scalability and maintainability.

---

##  Key Features

- **User Management**
  - User registration and authentication
  - Role-based access (Admin / User)

- **Project Management**
  - Create, update, and delete projects
  - View all projects in a structured way

- **Task Management**
  - Create and assign tasks
  - Update task status (To Do / In Progress / Completed)
  - Set deadlines and priorities

- **Task Tracking**
  - Track progress of tasks in real time
  - View tasks by project or user

- **Modern UI**
  - Clean and responsive user interface
  - Easy navigation and user-friendly design
